Obedient Cat (picoCTF 40)

Challenge Author: SYREAL

Category: General Skills

Difficulty: Easy

Description

This file has a flag in plain sight (aka ‘in-the-clear’)

Process / Notes

  1. wget the flag
  2. cat the flag file - and it’s there!

1 minutes 24 seconds to complete

Hints

Core Lessons

  1. Understand how to retrieve a file with wget
  2. Understand how to read a file’s contents with cat
